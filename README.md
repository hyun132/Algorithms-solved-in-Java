## 기본기능
#### 주택 별 매매 및 전,월세 거래내역 정보와 아파트 정보를 검색하는 기능 구현
#####    요구사항 : 아파트/주택 매매/전,월세 거래 정보에 대한 법정동 또는 아파트 검색이 가능하다.
#####               검색된 아파트/주택에 대해 상세 정보를 확인할 수 있다.
